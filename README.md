# AI-Web-App
Writeless AI is a web app which generates essays based on three things and they are as follows:

1- Title
2- Abstract
3- Citations

# Tech Stack
Bubble.io
RapidAPI Integrations:
1- AI CONTENT DETECTOR API (GPT4)
2- Paraphrase Genius API
